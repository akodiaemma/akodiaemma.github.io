⚡ One-time Firebase setup required

  1. Go to console.firebase.google.com → your project (or create one, it's free)
  2. Click Add app → Web, register the app, copy the firebaseConfig object
  3. Go to Firestore Database → Create database → start in test mode
  4. Paste your config into the Settings panel (bottom of sidebar) and click Connect
  5. That's it — hit Publish and your live page updates instantly for everyone 🎉
